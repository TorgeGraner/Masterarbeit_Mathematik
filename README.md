# Eigenschaften der Menge der H-Kobordismus-Klassen von Homotopiesphären
Ein (bis auf niedrige Dimensionen) vollständiger Beweis der Endlichkeit der Kervaire-Milnor-Gruppe $\Theta_k$ der Gruppe der H-Kobordismus-Klassen von Homotopiesphären durch Methodiken der Chirurgie-Theorie.

Ein besonderer Fokus liegt auf die Vermeidung von glatten Mannigfaltigkeiten mit Ecken, die sich zum Beispiel als kartesisches Produkt $\mathcal{M}\times\mathbb{I}$ ergeben, wobei $\mathcal{M}$ eine glatte Mannigfaltigkeit mit nicht-leerem Rand, und $\mathbb{I}$ das Einheitsintervall ist.
